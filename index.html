⚡ COMO JOGAR

🎯 OBJETIVO

Mova suas CyberBeasts pelo tabuleiro e invada o Core inimigo (nodo rosa no topo) para vencer!

📋 TURNO DO JOGADOR

1. Spawnar: Clique em uma unidade do banco (embaixo) para colocá-la em um Portal de Entrada.

2. Mover: Clique em uma unidade no tabuleiro, depois clique em um nodo verde para mover.

3. Próximo Turno: Clique no botão para a IA jogar.

⚔️ COMBATE

Quando sua unidade se move para um nodo com inimigo, a Data Wheel gira para ambos. O resultado decide quem vence!

🔵 Azul = Defesa   🟡 Ouro = Prioridade   🟣 Roxo = Glitch   ⚪ Branco = Normal   🔴 Vermelho = Miss

💀 DATA CENTER

Unidades eliminadas vão para o Data Center (2 slots no canto). Quando o 3º slot é ocupado, a mais antiga reboota de volta ao banco!

🃏 PLACAS

Overclock: +1 MP para todas as unidades em campo

Data Repair: Retorna uma unidade ao banco

Firewall: Bloqueia um nodo aleatório por 1 turno

Banco de Unidades (Clique para spawnar) 3 unidade(s) no banco
